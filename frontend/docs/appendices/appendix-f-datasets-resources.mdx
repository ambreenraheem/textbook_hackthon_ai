# Appendix F: Datasets and Learning Resources

Curated collection of datasets for robot learning and external educational resources.

## Robotics Datasets

### Manipulation Datasets

- **YCB Object and Model Set**: Household objects with 3D models and textures
  - URL: https://www.ycbbenchmarks.com/
  - Use: Grasping, manipulation, object recognition

- **EGAD! (Grasping Dataset)**: Adversarial grasp evaluations
  - URL: https://sites.google.com/view/egad-dataset
  - Use: Grasp planning, evaluation

- **Google Scanned Objects**: 1000+ scanned household items
  - URL: https://app.ignitionrobotics.org/GoogleResearch/fuel/collections/Google%20Scanned%20Objects
  - Use: Simulation, recognition

### Navigation Datasets

- **KITTI Vision Benchmark**: Autonomous driving data
  - URL: http://www.cvlibs.net/datasets/kitti/
  - Use: SLAM, odometry, object detection

- **TUM RGB-D Dataset**: Indoor RGB-D SLAM benchmarks
  - URL: https://vision.in.tum.de/data/datasets/rgbd-dataset
  - Use: Visual SLAM, 3D reconstruction

### Human-Robot Interaction

- **Berkeley Autolab Dexterity Dataset (BADD)**: Dexterous manipulation
  - URL: https://berkeleyautomation.github.io/dex-net/
  - Use: Learning-based grasping

## Pre-trained Models

### Computer Vision

- **YOLO (You Only Look Once)**: Real-time object detection
- **MiDaS**: Monocular depth estimation
- **Detectron2**: Facebook AI object detection and segmentation

### Natural Language Processing

- **GPT Models**: Text generation and understanding
- **BERT**: Bidirectional language representations
- **CLIP**: Vision-language models

## Learning Resources

### Online Courses

- **Coursera - Modern Robotics**: Kinematics and control
- **edX - Robotics MicroMasters (UPenn)**: Comprehensive robotics curriculum
- **Udacity - Robotics Nanodegree**: Practical robotic systems

### Books

- "Modern Robotics" by Lynch and Park
- "Probabilistic Robotics" by Thrun, Burgard, and Fox
- "Robotics, Vision and Control" by Peter Corke

### Research Venues

- **ICRA**: International Conference on Robotics and Automation
